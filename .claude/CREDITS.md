## When adapting more

When pulling a skill or pattern from another public repo:
1. Check the LICENSE file. MIT/Apache/BSD = reuse freely with attribution. CC BY-NC = paraphrase only (commercial-use restriction). No license = paraphrase only (all rights reserved default).
2. Add a line to this file and a header comment in the adapted file.
3. If paraphrasing a non-open-licensed source, rewrite substantially — don't copy verbatim.
