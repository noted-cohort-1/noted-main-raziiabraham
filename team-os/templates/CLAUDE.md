# Templates

Copy-paste starting points. Use the slash commands below if Claude Code is open — they scaffold + fill out for you.

| Template | Command | What it starts |
|---|---|---|
| `prd.md` | `/prd-new` | A new PRD |
| `feature-dossier/` | *(manual)* | A new `team-os/features/<slug>/` folder |
| `market-pulse.md` | `/market-scan` | A new market pulse in `research/market-pulse/` |
| `ship-log-entry.md` | `/ship-log` | A new line in a feature's `ship-log.md` |
| `competitor-tldr.md` | *(manual)* | A new `research/competitors/<slug>/tldr.md` |

When adding a new template, also add a row above and (ideally) a slash command so users don't have to remember file paths.
