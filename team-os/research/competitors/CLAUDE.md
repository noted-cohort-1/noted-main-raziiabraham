# Competitors

One folder per competitor we actively track. Start with `tldr.md` for each. Add `pricing.md`, `teardown.md`, `signals.md` only when they earn their place.

## Who we track (suggested starting set)

- **Notion** — the category leader. Always tracked.
- **Obsidian** — the local-first power-user alternative.
- **Craft** — design-led, Apple-centric.
- **Mem** — AI-native, acquired-and-folded narrative.
- **Reflect** — AI-native with backlinks.
- **NotionAI / Anthropic-Claude-for-docs / OpenAI Canvas** — AI features from the big platforms.

Add others as they become relevant.

## What to fill in first

For any competitor, start with:

```
competitors/<slug>/tldr.md
```

A `tldr.md` is 100–300 words: what the product is, target user, key differentiator, pricing in one line, link to their site. If the competitor matters enough to dig deeper, add `pricing.md` and `teardown.md` later.
